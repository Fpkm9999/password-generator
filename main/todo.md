# 랜덤 비밀번호 생성기

### 구현 목표
- [ ] Math.random()을 이용하여 랜덤한 난수 생성:
> "Math.random() 함수를 사용해 무작위 비밀번호를 생성하는 로직 개발."

- [ ] 사용자가 생성된 비밀번호의 수정을 하지 못하도록 막기:
> "생성된 비밀번호를 사용자가 직접 수정할 수 없도록 설계."

-[ ] 원하는 비밀번호 자릿수를 선택할 수 있는 옵션 추가:
> "사용자가 원하는 비밀번호의 길이를 선택할 수 있는 기능 구현."
